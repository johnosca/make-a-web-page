The Basis of the Web and HTML

The Web Consists of:

HTML- HyperText Markup Language: a set of standards, a variety of SGML, used to tag the elements of a hypertext document. It is the standard protocol for formatting and displaying documents on the World Wide Web.
URL-Uniform Resource Locator: a protocol for specifying addresses on the Internet.
HTTP-hypertext transfer protocol: the standard protocol for transferring hypertext documents on the World Wide Web.

HTML and CSS
HTML has a tree like structure.
HTML and CSS are both languages.
HTML controls the structure of a webpage.
CSS controls the style of a webapage.
When programmers talk about the DOM (Document Object Model) They're talking about the tree like structure of a page.

Developer Tools (in the browser)
HTML elements are either inline or block. Block elements form an invisible box around the content inside of them.

The Tree Like Structure of HTML:
The tree like structure comes from the fact that HTML elements can have other elements inside of them. You can draw this relationship like a family tree. My mother had multiple children. So did her mother and so on. In a browser this structure shows up in a series of nested boxes, inside of boxes and so on.

The Relationship Between Indented HTML and Boxes
When you read an HTML document as text you see a wave of changing indentations going up and down the left side of the document. The more indented and element is, the more deeply nested it's corresponding box is.

Text editors for programming
When writing code, programmers use special text editors (like Sublime Text for example). These editors make the programmers life easier. For example, some text editors will automatically generate a closing HTML tag when writing an opening tag.
